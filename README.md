# CyberAman - منصة الأمان الرقمي التونسية

خلّيك آمن على الإنترنت

## نظرة عامة

CyberAman منصة تثقيفية مجانية موجهة للجمهور التونسي لتعليمهم كيفية الحماية من:
- الاحتيالات الإلكترونية والتصيد الاحتيالي (Phishing)
- الأرقام الوهمية على التطبيقات
- سرقة الحسابات على وسائل التواصل
- المواقع الوهمية والمتاجر الإلكترونية المزيفة

## الميزات الرئيسية

### 1. الصفحة الرئيسية (Home)
- عرض سريع لأهم الهجمات الإلكترونية الشائعة
- ثلاث مسارات رئيسية: التحقق، المحاكاة، التعلم
- نصيحة أمان يومية

### 2. كاشف الاحتيالات (Detector)
- فحص النصوص والروابط لتحديد ما إذا كانت آمنة أم مريبة
- نظام تحليل يعطي درجة خطورة من 0 إلى 100
- توصيات فورية بناءً على النتائج

### 3. محاكي الهجوم (Simulator)
- سيناريوهات واقعية للتعرف على الهجمات الإلكترونية
- أسئلة تفاعلية: هل هذا آمن أم خطير؟
- نقاط وتعليقات فورية

### 4. الأكاديمية (Academy)
- دروس تفاعلية حول الأمان الرقمي
- اختبارات قصيرة بعد كل درس
- تتبع التقدم والشارات

### 5. اختبار الأمان الشخصي (Assessment)
- 10 أسئلة لتقييم مستوى حماية حسابك
- درجة أمان من ضعيف إلى قوي
- توصيات مخصصة بناءً على الإجابات

### 6. دليل التعافي (Recovery)
- خطوات سريعة إذا كنت ضحية احتيال
- جهات اتصال طوارئ محلية
- إرشادات فورية لتقليل الضرر

### 7. القاموس (Glossary)
- شرح المصطلحات الأمنية بلغة بسيطة
- أمثلة تونسية محلية
- محرك بحث للمصطلحات

### 8. الإبلاغ عن الاحتيالات (Report)
- نموذج لتقديم بلاغات عن الاحتيالات
- حماية المجتمع بمشاركة التجارب

### 9. ملفي الشخصي (Profile)
- تتبع النقاط والمستويات
- الشارات والإنجازات
- إحصائيات التقدم

## المميزات التقنية

### التصميم
- RTL (من اليمين إلى اليسار) لدعم اللغة العربية
- تصميم متجاوب يعمل على جميع الأجهزة
- وضع ليلي ووضع كبار السن

### الأمان والخصوصية
- بدون تسجيل إجباري
- البيانات تُخزن محلياً في localStorage
- لا يتم جمع بيانات شخصية

### الإمكانية
- وضع كبار السن مع أزرار وخطوط أكبر
- لغة عربية تونسية (دارجة) واضحة وبسيطة
- بدون جargon تقني معقد

### الألعاب والتحفيز
- نظام نقاط وتسويات
- شارات قابلة للكسب
- مستويات لتتبع التقدم

## المتطلبات التقنية

- Node.js 18+
- pnpm

## التثبيت والتطوير

### 1. استنساخ المستودع
```bash
git clone <repo-url>
cd cyberaman
```

### 2. تثبيت المتعلقات
```bash
pnpm install
```

### 3. التطوير المحلي
```bash
pnpm dev
```

الموقع سيكون متاحاً على `http://localhost:3000`
